Introduction
==========================================


Welcome to the OORT Documentation.
This container image offers a streamlined and optimized Docker base image tailored for
running various PHP applications with ease and efficiency. It supports powerful
frameworks like Laravel Octane, Laravel Horizon, Laravel Reverb, Laravel Pulse,
Laravel Scheduler, as well as Symfony components such as Symfony Runner,
Symfony Messenger, and Symfony Scheduler. Designed to be lightweight, this image
ensures fast performance and minimal resource usage, making it an ideal foundation
for PHP applications in modern, scalable environments.
